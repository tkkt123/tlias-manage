# Tlias 智能学习平台管理系统

> 基于 Spring Boot + Vue 的前后端分离后台管理系统，在黑马课程项目基础上进行了全面的技术升级与功能增强。

---

## 🚀 项目简介

这是一个为教育机构设计的智能学习平台管理系统，源于对黑马程序员课程的学习与实践。**我在掌握核心思路的基础上，对技术架构和业务功能进行了深度改造与创新。**

**🎯 核心改进与差异化：**
- 🔄 **架构升级**：从传统 JavaWeb 架构升级为 **Spring Boot + Vue** 前后端分离架构
- 📦 **存储方案**：将原项目的阿里云 OSS 替换为 **Minio**，实现私有化文件存储
- ✨ **功能增强**：独立设计并实现了**用户密码修改、数据统计**等核心业务功能
- 🛡 **安全加固**：集成 **JWT** 令牌机制，实现安全的用户认证与授权

## 🛠 技术栈

**后端技术：**
- **框架**：Spring Boot, Spring MVC, MyBatis
- **安全**：JWT Token 认证
- **存储**：MySQL, **Minio** (对象存储)
- **工具**：Maven, JDK 17

**前端技术：**
- **框架**：Vue 3, Element Plus
- **工程化**：Nginx (反向代理与静态资源部署)
- **基础**：HTML5, CSS3, JavaScript, Ajax

**开发工具：**
- IDEA, VSCode, Git, Maven

## 🚀 快速开始

### 环境要求
- JDK 17+
- MySQL 8.0+
- Maven 3.9+
- **Minio Server** (对象存储服务)

### 部署步骤

1. **数据库初始化**
    - 创建数据库 `tlias`
    - 执行 `help/tlias.sql` 文件初始化表结构

2. **Minio配置**
    - 启动 Minio 服务
    - 创建存储桶并配置访问权限
      （可根据“help/docker-compose.yml”快速搭建minio服务）

3. **应用配置**
    - 修改 `src/main/resources/application.yml` 中的配置：
        - 数据库连接信息
        - **Minio连接配置**

4. **启动项目**
    - 后端：找到主启动类 `TliasWebManagementApplication`点击运行按钮 ▶️ 即可
    - 前端：使用 `npm run dev` 启动Vue开发服务器
    - 后端访问：`http://localhost:8080`
