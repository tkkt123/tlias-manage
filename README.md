# tlias-manage
用于学习JavaWeb的实战项目,供大家参考

## 项目简介
本项目是一个基于 JavaWeb 技术栈的后台管理系统，源于对黑马程序员课程的学习与实践。**我在掌握核心思路的基础上，对技术选型和功能模块进行了个性化改造与增强。**

**与原课程项目的核心差异：**
- 🔄 **对象存储方案**：将原项目的阿里云 OSS 替换为 **Minio**，实现了自建私有云存储。
- ✨ **新增功能**：独立设计并实现了**用户修改密码**功能。


## 技术栈
- **后端**： Jwt, MyBatis, MySQL, springBoot, minio
- **前端**：HTML, CSS, JavaScript, vue, Ajax, nginx, Element Plus
- **开发工具**：IDEA, Maven, Git,VsCode

## 功能特性
- 用户登录、更改密码
- 部门管理：增、删、改、查
- 员工管理：增、删、改、查、分页显示
- 课程管理：增、删、改、查、分页显示
- 学员管理：增、删、改、查、分页显示
- 条件查询：根据姓名、部门等筛选员工、学员、班级
- 数据统计：按照某一特性统计员工和学员的数量

## 快速开始
1.  **环境要求**：JDK 17, MySQL 8.0, spring3.4.6, Maven 3.9.9
2.  **数据库配置**：
    - 创建数据库 `your_database`。
    - 执行 `sql/init.sql` 文件初始化表和数据。
    - 修改 `src/main/resources/db.properties` 中的数据库连接信息。
3.  **部署运行**：
    - 使用 Maven 打包项目：`mvn clean package`
    - 启动 SpringBoot项目，访问 `http://localhost:8080`
